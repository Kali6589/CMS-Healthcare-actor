# CMS Healthcare Data Lookup

Look up Medicare reimbursement rates, decode claim denial codes, and search the NPI provider registry.

## Three tools in one Actor

### 1. Medicare Fee Schedule
CPT code + state → Medicare rates for every locality

### 2. Denial Code Lookup (CARC/RARC)
Denial code → description + recommended action

### 3. NPI Provider Search
8M+ US providers by NPI, name, specialty, state

## Pricing
$5 per 1,000 results

## Data sources
- CMS: pfs.data.cms.gov
- NPI: npiregistry.cms.hhs.gov
- X12: x12.org
